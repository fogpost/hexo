# 开头
大部分人在学习逆向时老是利用以及存在的软件或是利用题目去进行学习这是很好的方式但我们也需要多去分析自己的代码，来加强我们对一些特定函数的段落的分析。

## 源码
```c
#include<stdio.h>
int main(){
	int a,flag=0;
	scanf("%d",&a);
	int j,k,l;
	j=a/100;
	k=(a/10)%10;
	l=a%10;
	if(j==k)flag++; //判断百位十位相同
	if(k==l)flag++; // 判断十位个位相同
	if(j==l)flag++; //判断百位各位相同
	if(flag==1)printf("Yes");
	else printf("No");
	return 0;
}
```
