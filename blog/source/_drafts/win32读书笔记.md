# 80X86三种工作模式（32位处理器）
>实模式（DOS）  
>低32位*16做基地址+偏移地址=真实物理地址，不能内存分页，可读可写可执行。特权级,不支持硬件多任务  
>四字节中断向量，32位寄存器，FS、GS、辅助达到6个可访问段  
>保护模式（windows）  
>32位寻址达到4G，支持内存分页，虚拟内存，硬件多任务切换，保护处理器自处理  ，优先级机制
>虚拟8086模式    
>任务形式上的保护模式，任务切换和内存分页，8086寻址虚拟地址映射，有保护异常的虚拟特权指令
