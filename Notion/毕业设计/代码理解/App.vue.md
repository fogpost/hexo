
## 核心模块
### **Header（顶部工具栏）**
- **启动/停止后端**：控制后端服务
- **启动/停止抓包**：控制网络数据包捕获
- **刷新检测**：重新分析流量
- **状态指示**：显示后端和抓包的运行状态（绿点=运行，红点=停止）
### **Sidebar（左侧边栏）**

| 卡片       | 功能                       |
| -------- | ------------------------ |
| **运行状态** | 显示消息、项目路径、数据目录；自动刷新/轮询开关 |
| **抓包网卡** | 选择网络接口，显示当前选中网卡          |
| **检测结果** | 统计包数、告警数、源/目的IP数、协议分布    |
| **告警列表** | 显示检测到的安全告警，可展开查看证据并定位包   |
### **Main Panel（主面板）**
#### **上方控制栏**
- 刷新列表 / 导包 / 文件列表切换 / 解析下一个 / 删除包
- 显示当前文件名和总包数
#### **左侧：数据包列表**

- 支持文件列表面板（可折叠）显示所有PCAP文件
#### **右侧：数据包详情**
- **解析数据**：分层显示网络协议栈（L2/L3/L4等）
- **Hex视图**：原始十六进制数据，鼠标悬停高亮对应字节

### **Terminal Center（终端中心）**
- **本地终端**：创建本地Shell会话
- **SSH终端**：支持密码/RSA密钥认证远程连接
- **终端列表**：显示所有活跃终端，可切换
- **xterm.js集成**：完整的终端模拟器

## **自动轮询机制** (`pollingTick`)

### **Hex高亮交互**

- 悬停协议字段 → Hex中对应字节高亮
- 鼠标事件触发 `highlightField(offset, length)`
- 高亮范围计算：`i >= highlightStart && i < highlightStart + highlightLength`

### **终端管理**

- 维护多个终端历史（Map结构）
- 支持UTF-8/GB18030编码自动切换
- 限制历史大小为500KB防止内存溢出
- Wails运行时数据交互（桌面应用功能）

### **文件处理**

- 导入PCAP文件：FormData + `POST /upload-pcap`
- 删除文件：`DELETE /data-file`
- 自动解析Live文件夹中的最新文件

## 生命周期
onMounted  → init() → 初始化所有数据，启动轮询
watch(activeTerminalId) → 切换终端时重新渲染
onUnmounted → 清理轮询、事件监听、终端资源
