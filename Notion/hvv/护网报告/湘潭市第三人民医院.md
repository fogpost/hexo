扫描可能出现的路径
```
dirsearch -u http://target.com -e * -o dir.json -f json
```
过滤有效结果
```
cat dir.json | jq '.results[] | select(.status == 200 or .status == 302) | .path'
```
提取敏感目录
```
cat dir.json | jq '.results[] | select(.path | test("admin|login|api|cms|manage|upload"))'
```
导出干净列表
```
cat dir.json | jq -r '.results[] | select(.status==200) | .url' > alive.txt
```
