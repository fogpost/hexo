常见 Web 漏洞主要包括 SQL 注入、XSS、CSRF、文件上传漏洞以及反序列化漏洞。SQL 注入主要由于 SQL 拼接导致数据库信息泄露；XSS 是攻击者向网页注入恶意脚本在用户浏览器执行；CSRF 利用用户登录状态伪造请求；文件上传漏洞可以上传 WebShell 获取服务器权限；反序列化漏洞则通过构造恶意对象触发程序执行危险代码。

SSRF 是服务器端请求伪造漏洞，攻击者通过控制服务器请求的 URL，使服务器访问内部网络或敏感资源，例如访问 127.0.0.1 或云 metadata 服务。攻击者可以利用 SSRF 扫描内网端口、读取云凭证甚至攻击内部服务。防御方法包括 URL 白名单、禁止访问内网地址以及限制协议类型。
