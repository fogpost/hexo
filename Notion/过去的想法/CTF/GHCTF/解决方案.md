如果你要 **精确匹配 HTTP 传输的 URL 编码**，建议使用 `**rawurlencode()**` 进行两次编码，而不是 `urlencode()`：


```php
<?php
$original = "POST /Manage HTTP/1.1\n".
            "host:127.0.0.1\n".
            "Content-Type:application/x-www-form-urlencoded\n".
            "Content-Length:7\n\n".
            "cmd=env";

// 第一次编码
$encoded_once = rawurlencode($original);

// 第二次编码
$encoded_twice = rawurlencode($encoded_once);

echo $encoded_twice;
?>

```
