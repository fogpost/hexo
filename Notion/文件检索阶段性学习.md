## 🗂️ 阶段 1：打牢 Django 基础（第 1–2 周）

### 🎯 目标

- 掌握 Django 项目结构、模型、视图、模板等基础
- 学会使用 Django Admin 管理数据
- 搭建用户注册 / 登录流程

### 📚 重点内容

| 内容 | 说明 |
| --- | --- |
| Django 项目结构 | `manage.py`、`settings.py`、`urls.py` |
| MVT 模型 | 模型（Model）、视图（View）、模板（Template） |
| 表单系统 | `ModelForm` 创建/编辑对象 |
| 用户认证系统 | 注册、登录、注销 |
| Django Admin | 自定义模型显示与管理 |

### 🛠 实践任务

- 创建 Django 项目和基础 app（如 `accounts`, `documents`）
- 实现用户注册 / 登录 / 登出 / 修改密码功能
- 实现发布“文章”功能（模拟文档），并在 Admin 后台管理

---

## 🗂️ 阶段 2：核心功能实现（第 3–4 周）

### 🎯 目标

- 完成文档上传、检索、权限管理、版本控制等核心模块
- 学会使用 Django 的自定义用户模型（AbstractUser）

### 📚 重点内容

| 内容 | 说明 |
| --- | --- |
| AbstractUser | 自定义用户模型，加入等级字段等 |
| 文件上传 | `FileField`, `MEDIA_ROOT`, 预览/下载 |
| 权限控制 | `@login_required`、用户等级、访问限制 |
| 模型继承 | 文档版本控制（多个版本的文档关系） |
| CKEditor 富文本 | 文档内容编辑器集成 |

### 🛠 实践任务

- 创建自定义用户模型并替换默认用户
- 实现文档上传、编辑、删除、预览功能
- 为不同用户设置文档访问限制（公开/私有）
- 集成 CKEditor 富文本编辑器用于文档模板

---

## 🗂️ 阶段 3：高级功能扩展（第 5–6 周）

### 🎯 目标

- 增加全文检索、协作共享、数据统计等增强功能
- 记录用户操作日志，完善系统体验

### 📚 重点内容

| 内容 | 说明 |
| --- | --- |
| Whoosh + Haystack | 全文检索实现 |
| 共享机制 | 多用户文档共享，访问控制 |
| Django Signals | 记录日志、触发备份等事件 |
| Chart.js 数据可视化 | 上传趋势、用户活跃等图表 |
| 多对多关系 | 文档与共享用户的关系建模 |

### 🛠 实践任务

- 为文档添加关键词搜索和全文搜索功能
- 实现用户之间的文档共享（协作访问）
- 使用 Chart.js 实现文档数量趋势图表
- 创建操作日志（如：用户何时查看了文档）

---

## 🗂️ 阶段 4：部署与备份上线（第 7–8 周）

### 🎯 目标

- 完成系统部署、备份功能、上线发布
- 实现系统配置与容灾备份策略

### 📚 重点内容

| 内容 | 说明 |
| --- | --- |
| Docker | Django + Gunicorn + Nginx 容器化部署 |
| 数据备份 | SQLite 文件打包、手动或定时导出 |
| Nginx 配置 | 服务代理、静态资源处理 |
| 安全策略 | 文件权限、静态资源隔离、用户数据保护 |

### 🛠 实践任务

- 使用 `Docker` 容器化 Django 应用
- 将文档和数据库打包为 `.zip` 备份文件
- 实现一键导入历史快照恢复功能
- 搭建生产环境（可部署到阿里云/腾讯云/本地）
