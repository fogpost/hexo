## 2️⃣ 文件上传（非常高命中🔥）
测试：
- 上传 php / jsp / aspx
- 绕过：
    - 双扩展名：`1.php.jpg`
    - MIME绕过
    - 大小写绕过
👉 EDU 很多系统：  
✔ 没做校验  
✔ 可以直接 getshell
---

- sqlmap（辅助，不要全自动依赖）

## 4️⃣ 越权漏洞（EDU高爆率🔥）
测试：
- 改参数 ID
?id=1001 → ?id=1002
👉 看：
- 能不能看到别人信息
- 能不能修改别人数据

## 7️⃣ SSRF / 内网探测（进阶）
目标：
- 打内网服务

```
1. 收集子域名
2. 逐个访问
3. 用 Burp 抓包
4. 找功能点
5. 重点测：
   - 登录
   - 上传
   - 查询接口
6. 手动改参数（越权）
7. 最后再用自动化工具补扫
```
